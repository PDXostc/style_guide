Modifiers

Modifiers are classes that can added to elements to quickly override default values.

Class Description
.hidden Sets the display to none
.borderless Removes any border properties
.z-layer Layers any element to 100% of the first positioned anscestor
.no-margin Sets the margin to 0
.left Floats the element to the left
.right Floats the element to the right
.left-align Aligns the text of an element to the left
.center Aligns the text of an element to the center
.right-align Aligns the text of an element to the right
.flex-parent Enables flexbox for a parent element
.flex-item Sets the flex-grow to 1
.flex-item-2 Sets the flex-grow to 2
.flex-item-3 Sets the flex-grow to 3
.flex-column Sets the flex parent to flow children vertically
.flex-row Sets the flex parent to flow children horozontally
Only needed to override flex-direction: flex-column
.flex-center Centers a child flex element vertically and horizontally
.flex-column-center Sets the flex parent to flow AND center its children vertically
.space-between Sets the flex parent to distribute its children evenly spaced from end-to-end
