Boxes

"Boxes" are sections of content that can be individually styled. Applying a box style on a parent container will allow for easier styling of enclosed headers, headings, text, backgrounds, etc.

Box variations:

  • .box-outline
  • .box-infill
  • .box-inshadow
  • .box-outshadow

These variation styles should be used on the parent container of your content. If you have header content specific to your created section, you could use the .box-header class which applies a flush header area to the top of the box.
