Buttons

Not fully implemented in common.css

These buttons provide a UI for the user to initiate actions such as submitting a form or confirming a choice.

Button class and button elements

Button Class

The .button- classes allow for button styling on any element including the button element and input type="button" element.

Use classes on non-button elements when you want the appearance of a button but aren't submitting any information. For example, you can use a button class to style an anchor link.

Button Element

The button element should be used exclusively within the form element for submitting information to the application.
