These buttons provide a UI for the user to initiate actions such as submitting a form or confirming a choice.

Button class vs. Button element

Button Class

The .button- classes allow for button styling on elements other than the button element. This is particularly useful for styling anchor links as buttons.

Button Element

The button element should be used exclusively within the form element for triggering actions.

General Rule

Use the button element within the form element when submitting user input and triggering Javascript.

Use .button- classes on links (if desired) and on other elements that do not trigger actions but where you still desire button styling.
