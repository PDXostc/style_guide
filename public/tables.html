Tables ▲

Tables provide an interface for displaying tabular data to the user in a grid. Such data should necessitate the use of columns, rows and possibly headings. Examples of tabular data include media player lists, gathered statistics, and contact lists. Tables should not be used purely as a means to layout document content or for simple lists.

Flex-Based Column Widths

Column widths are set by way of the flex property. This allows for control over column width versus dynamic sizing based on text content.

The default flex settings for the table element have been set to:

  • flex: 2 for the first column
  • flex: 1 for the remaining columns

In the example above, the column widths are calculated like this:

Using a Table Class

Create a class to apply to the table element to allow for easier selection and styling of table elements.

Custom Column Widths

To customize column widths, use :pseudo selectors to target rows by position, index or occurence. Then set the flex property as desired.

Custom Row Styles

To select alternating rows, use the :nth-child(even) and :nth-child(odd) :pseudo selectors.

HTML Symbols: To add an HTML symbol to a cell, add a class to the cell and use the :before :pseudo selector and content property. Symbols in CSS must use their Hex reference and be escaped with a leading "\".

// Selects the first cell in each row of the body of the table
// and applies a tinted empty star symbol
.playlist tbody td:first-of-type:before {
  content: "\2606";
  color: rgba(255,255,255,.2);
}

// Selects the first cell in each row with class "fav-song"
// and applies a solid orange star symbol
.playlist td:first-of-type.fav-song:before {
  content: "\2605";
  color: #fe9c00;
}

Background Images: To add a background image to a cell, apply a class and use the background-image property. Be sure to override any content symbols being added.

// Adds a background gif to the cells with the class "playing" and "dancing"
.playing {
  background-image: url(../images/eq.gif);
}

.dancing {
  background-image: url(../images/banana.gif);
  background-size: auto 100%;
}

.playing, .dancing {
  background-repeat: no-repeat;
  background-position: left center;
}

// Overrides the default empty star from the previous example for cells with class "playing"
.playlist tbody td.dancing:before,
.playlist tbody td.playing:before {
  content: none;
}
