A flex primer

Flex allows child elements to expand in size based on a numerator. For instance, a flex parent element with two children each with a flex of 1 would look like this:

1
1

Each child element is 1 part of 2 or 1/2 of the parent.

If no flex value is set, the default value is 0 which would cause the element to collapse. If the element has content or set sizing, it'll take up only the space needed.

Flex for AGL

The #base element has 3 noteable CSS properties:

display: flex;
justify-content: space-between;
flex-direction: column;
 
display: flex;

...enables flex layout.

flex-direction: column;

...causes the flexing to happen vertically.

justify-content: space-between;

...causes the #top-bar and #bottom-bar to stay at the top and bottom without positioning.

#app has one important property:

flex: 1;

...which allows the #app container to expand to the remaining space.
