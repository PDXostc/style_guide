Sliders

These sliders provide a UI for sliding components that receive range inputs (such as volume) from the user. The .slider-input is an input element of type range.

Adding a Slider to HTML

  1. Copy the HTML of the desired slider (from the code snippets below)
  2. Place the HTML within a parent element with explicit height and/or width
    • The length of the slider is scaled to its parent
      • Vertical sliders need a parent with height
      • Horizontal sliders need a parent with width
  3. Add control buttons (as noted in the code snippets) if desired.
  4. To control the slider from Javascript, change the value attribute of the .slider-input.
  5. The CSS for this component is already included in common.css.
