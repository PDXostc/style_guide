Sliders

These sliders provide a UI for sliding components that receive scalable user inputs such as volume.

Input type: range

HTML5 provides a new input type, "range", which reduces the complexity of receiving scalable input from the user. In its simplest form, a range input looks like this:

<input type="range" />

The input can include the following useful attributes:

  • value: the initial value
  • name: the name of the input within the form
  • min: the minimum allowed value
  • max: the maximum allowed value
  • step: the intervals of the input (default is 1)

The range input is styled using a variety of pseudo-selectors:

  • ::-webkit-slider-thumb: Styles the slider handle
  • ::-webkit-slider-runnable-track: Styles the track that the slider moves within
  • ::-webkit-slider-thumb:after: Styles the color level bar adjacent to the slider handle

Using Sliders

Sliders come in vertical and horizontal formats:

.vertical-slider, .horizontal-slider

Sliders come in three preset weights:

Style Class Description
Standard Weight no class required Has a 40px track and an 80px thumb
Thin Weight .thin-slider Has a 20px track and an 40px thumb
Thick Weight .thick-slider Has a 60px track and an 120px thumb

To use the sliders you need a parent element with the slider format class and the slider weight class (if desired). Within the parent you need an input type="range" with the class .slider-input. For example, a horizontal thin slider would be structured like this:

<div class="horizontal-slider thin-slider">
  <input class="slider-input" type="range" value="50">
</div>
