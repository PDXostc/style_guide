HTML Guidelines

Javascript in HTML

  • HTML pages should not contain blocks of JavaScript code using thescript element.
  • The script element should only be used for attaching external JavaScript code.
  • HTML Elements should not contain event attributes (eg. onclick="someFunction")

General Formatting

  • Use a new line for every block, list, or table element, and indent every such child element.
  • Use 2 spaces for indentation.
  • When quoting attributes values, use double quotation marks.
