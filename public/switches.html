Switches

Switches provide an interface for receiving A/B type user input. Such user interactions include toggling settings and on/off states. The switch UI comes in three sizes:

.switch-large, .switch-small, .switch-xsmall

JS toggles the ".on" and ".off" classes of the ancestor ".switch".

".switch" has 2 descendants:
".switch-plate" which is the housing that changes color, and
".switch-control" which is the knob that toggles position.

The switch class is set to ".off" by default.
