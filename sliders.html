Sliders

These sliders provide a UI for sliding components that receive scalable user inputs such as volume. These sliders are currently non-functional.

Adding a Slider to HTML

  1. Copy the HTML of the desired slider (from the code snippets below)
  2. Place the HTML within a parent element with explicit height and/or width
    • The length of the slider is scaled to its parent
      • Vertical sliders need a parent with height
      • Horizontal sliders need a parent with width
  3. To control sliding from Javascript, change the flex-basis property of the .slider-level in percentage units.
  4. The CSS is already included in common.css
