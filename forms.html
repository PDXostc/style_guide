Text Inputs

Text inputs include: input[type="text"], input[type="password"], textarea

Stacked Label/Input

<label for="id-of-input"> ... </label>
<input id="id-of-input" class="text-input" size="45" placeholder="Placeholder text">

label for attribute: The for attribute binds the label to the associated input field. Its value should be equal to the value of the input id.

Horizontal Label/Input

The .input-group class allows grouping of label and input fields for better alignment of mutli-input forms.

Without grouping, input fields appear immediately after their label. And <br> tags are required to prevent labels and inputs from displaying inline.




With grouping—and the help of flex properties—input labels and fields align and <br> tags aren't needed.

Right-Aligned Labels

<div class="input-group">
  <label for="id-of-input"> ... </label>
  <input id="id-of-input" class="text-input" size="45" placeholder="Placeholder text">
</div>

label for attribute: The for attribute binds the label to the associated input field. Its value should be equal to the value of the input id.

input size attribute: The size attribute sets the character length (i.e. width) of an input field. It is not a required attribute but it does have a default value of 20. To standardize a width across all inputs, add a class with a width declaration to your app-specific CSS.

Left-Aligned Labels

<div class="input-group">
  <label for="id-of-input" class="left-label"> ... </label>
  <input id="id-of-input" class="text-input" size="45" placeholder="Placeholder text">
</div>

Labels in an .input-group appear aligned to the right. Alignment is handled through the flex property {justify-content: flex-end;}. To align labels to the left, apply the class "left-label" to the label elements.

Color Labels, Inputs, and :focus states

To draw attention to an input or differentiate it as important, use the following color classes on the label and input.

<form>

  <div class="input-group">
    <label for="color-input-yellow" class="yellow-lt">Yellow Input</label>
    <input type="text" class="text-input input-yellow" id="color-input-yellow">
  </div>

  <div class="input-group">
    <label for="color-input-blue" class="blue-viv">Blue Input</label>
    <input type="text" class="text-input input-blue" id="color-input-blue">
  </div>

  <div class="input-group">
    <label for="color-input-green" class="green-lt">Green Input</label>
    <input type="text" class="text-input input-green" id="color-input-green">
  </div>

  <div class="input-group">
    <label for="color-input-orange">Orange Input</label>
    <input type="text" class="text-input input-orange" id="color-input-orange">
  </div>

</form>

Default coloring of label text is orange. Default coloring of input text and :focus glow is white.
