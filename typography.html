Other Text Sizes

Below is a list of old common text-size variations and suggested replacements. When deciding how to size text, keep the following in mind:

  • Use the small element to reduce type. It is relative and will reduce font-size by approx. 1/6th
  • If you need a size not specified here, add a custom class to your app that is stand-alone or modifies and existing style, egs: 
    /* stand-alone class */
    .nav-large-text { ...

    /* modifier class */
    h4.nav { ...
  • Any heading should only appear once in any single element.
    • To achieve the appearance of a heading on sibling elements:
      • create a custom class in your app that matches the heading properties
      • apply the class to a non-heading text element, eg:
        • .matches-h1 {
  font-size: 7rem;
  color: #fc9e00;
}
