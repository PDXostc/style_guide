Structure ▲

This HTML/CSS structure for top-level container elements is partially available in the AGL App Suite. The App Suite does not yet have the recommended naming conventions or a container for modals. Refer to Settings Set-Up for the current implementation.

This HTML structure provides a flexible, layered layout that applies to all applications. It utilizes Flexbox Layout to limit the need for absolute positioning, floats, z-indexing, and absolute sizing.

The HTML

The HTML body is divided into 3 primary containers:

  • #base
  • #settings
  • #modal

The #base container contains:

  • #top-bar
  • #app
  • #bottom-bar

Each of these contain a .content container that serves as the target for loading app- or component-specific content.

Flex for AGL

The #base element has 3 noteable CSS properties:

display: flex;
justify-content: space-between;
flex-direction: column;
 
display: flex;

...enables flex layout.

flex-direction: column;

...causes the flexing to happen vertically.

justify-content: space-between;

...causes the #top-bar and #bottom-bar to stay at the top and bottom without positioning.

#app has one important property:

flex: 1;

...which allows the #app container to expand to the remaining space.

Settings Set-Up
  1. Create a div with id of "base" as the first child in the body
  2. Move #topBar and #bottomBar into it
  3. Create a div with id of "app" within #base
  4. Place the new #app div after #topBar but before #bottomBar
  5. Create a div with id of "clockElement" as the first child in #app
    • If you don't need the date and time in your app, comment out the #clockElement div
  6. Move all remaining app-specific containers into #app after #clockELement
    • Be careful to maintain the current order
    • You can wrap the content into another div for organization and code folding
  7. Add a div with id of "settingsPage" after #base
    • Give it a class of "settingsPage hidden" (Xwalk 12/2014)
    • Javascript will now only add the content of settings (not the parent div)

Your HTML should look similar to this:

<body>
<div id="base">
<div id="topBar"></div>
<div id="app">
<div id="clockElement"></div>
<-- Your app-specific HTML -->
</div>
<div id="bottomBar"></div>
</div>
<div id="settingsPage" class="settingsPage hidden"></div>
</body>
