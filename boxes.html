Boxes ▲

"Boxes" are sections of content that can be individually styled. Applying a box style on a parent container will allow for easier styling of enclosed headers, headings, text, backgrounds, etc.

Boxes come in three styles:

  • .box-outline
  • .box-inshadow
  • .box-outshadow

These styles should be used on the parent container of your content. If you have header content specific to your created section, you could use the .box-header class which applies a flush header area to the top of the box.

// box-outline with box-header and box-content example
<div class="box-outline">
  <header class="box-header">
    <h5>box-outline</h5>
  </header>
  <div class="box-content">
    <p><small>Lorem ipsum dolor sit amet, consectetur</small></p>
  </div>
</div>

// box-inshadow example
<div class="box-inshadow">
  <h5>box-inshadow</h5>
  <p><small>Lorem ipsum dolor sit amet, consectetur</small></p>
</div>

// box-outshadow example
<div class="box-outshadow">
  <h5>box-outshadow</h5>
  <p><small>Lorem ipsum dolor sit amet, consectetur</small></p>
</div>
